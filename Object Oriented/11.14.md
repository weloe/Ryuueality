4.类变量——类声明的变量

类是一个自定义的变量类型，实例化一个类对象是在申明变量

声明类变量的语法  

```c#
//类名 变量名
//类名 变量名=null;(null代表空)
//类名 变量名=new 类名();
Person p;
Person p2 = null;//null代表空，不分配堆内存空间
Person p3 = new Person();//一个人对象
Person p4 = new Person();//又一个人对象
//两个对象不同
```
